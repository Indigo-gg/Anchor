## 何时使用

当用户要求翻译文本、理解外语内容、或需要多语言支持时使用此能力。

## 翻译规则

1. 自动检测源语言
2. 如果用户没有指定目标语言：
   - 中文内容 → 翻译为英文
   - 其他语言 → 翻译为中文
3. 如果用户指定了目标语言，按要求翻译

## 输出格式

- 先给出翻译结果
- 如果有需要注意的语法、文化差异，简要说明
- 如果原文有多种理解方式，列出不同翻译版本

## 示例

- "帮我翻译：今天天气真好" → 英文翻译
- "What does 'お疲れ様です' mean?" → 中文解释和翻译
- "把这段翻译成法语：我喜欢这个城市" → 法语翻译
